\subsection{Detailed Description}
An object containing a list of candidates. 


\begin{DoxyItemize}
\item The Candidae\+List class will be the main structure in which Ballots are stored.
\item The \hyperlink{class_candidate_list}{Candidate\+List} stores an array of \hyperlink{class_candidate}{Candidate} classes. It then has options to add, and remove the Ballots from this array
\item It also has a function that will return the \hyperlink{class_candidate}{Candidate} who has the lowest Ballots. 
\end{DoxyItemize}
